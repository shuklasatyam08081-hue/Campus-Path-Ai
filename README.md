# 🦾 CampusPath AI — Developer Career & Collaboration Platform

![Tech Stack](https://img.shields.io/badge/Stack-MERN--AI-emerald?style=for-the-badge)
![UI Aesthetic](https://img.shields.io/badge/Aesthetic-GitHub--Minimalist-gray?style=for-the-badge)

**CampusPath AI** is a comprehensive, AI-powered career acceleration and collaboration platform designed for software engineers. It architecturally maps your learning trajectory using Gemini AI, verifies your technical DNA via GitHub, provides real-time collaborative Focus Rooms, and helps you build professional, data-driven portfolios to land your dream role.

---

## ✨ Core Features

### 🗺️ AI-Powered Learning Roadmaps
- **Dynamic Trajectories:** Personalized learning paths generated by Gemini AI tailored to your target role.
- **3D Skill DNA:** Interactive, Three.js-powered 3D visualization of your curriculum milestones.
- **Verified Progress:** Automated repository auditing to verify task completion and unlock achievements.

### 🎨 Developer Portfolio Builder
- **Real-Time Engine:** Build your professional legacy in a live-preview studio editor.
- **GitHub-Inspired Themes:** Clean, syntax-themed, and professional minimalist layouts.
- **Data Injection:** Automatically integrate your GitHub stats, roadmap progress, and verified projects into a public profile.

### 🐙 GitHub Profile Analyzer & Repo Reviewer
- **Codebase Auditing:** Deep-dive analysis of your repositories to identify technical patterns.
- **Repo Auto-Reviewer:** Automated, URL-based code review and feedback for your repositories.
- **Skill Radar & Heatmaps:** Professional-grade visualization of technical proficiency and contribution frequency.

### 🤝 Collaboration & Growth
- **Focus Rooms:** Real-time, collaborative pomodoro study and work rooms powered by WebSockets.
- **Developer Community:** Integrated discussion boards and resource sharing for peer-to-peer growth.
- **Jobs Board:** Curated and dynamically filtered job listings to match your acquired skills.

---

## 🛠️ Technical Stack

### Frontend
- **React 19 & Vite:** Lightning-fast, modern reactive UI.
- **Tailwind CSS v4:** Clean, GitHub-inspired utility-first styling.
- **Framer Motion & Three.js:** Liquid page transitions, micro-animations, and 3D skill visualizers.
- **Recharts:** High-performance data visualization for GitHub stats.

### Backend
- **Node.js & Express:** Scalable API orchestration hub.
- **MongoDB & Mongoose:** High-speed document storage for user data, roadmaps, and rooms.
- **Google Gemini AI SDK:** Large Language Model integration for intelligent content generation.
- **Socket.io:** Real-time event streams for Focus Rooms and Community features.

## 🚀 Quick Launch Sequence

### 1. Repository Setup
```bash
git clone https://github.com/Shubham-k-yadav/project.git
cd project
```

### 2. Backend Environment
```bash
cd backend
npm install
# Create a .env file with:
# PORT=5000
# MONGO_URI=your_mongodb_uri
# GEMINI_API_KEY=your_key
# JWT_SECRET=your_secret
# GITHUB_CLIENT_ID=your_github_id
# GITHUB_CLIENT_SECRET=your_github_secret
npm run dev
```

### 3. Frontend Environment
```bash
cd frontend
npm install
npm run dev
```

## 📐 Design Philosophy: GitHub Minimalist
CampusPath AI leverages a clean, professional aesthetic optimized for developer workflows.
- **Syntax-Themed Components:** High-contrast, legible interfaces inspired by modern code editors.
- **Responsive Layouts:** Refined padding and max-width constraints for consistency across all screens.
- **Liquid UX:** Hardware-accelerated micro-interactions ensuring a butter-smooth feel without overwhelming the content.

---

## 🛡️ Security & Authentication
- **JWT-based Sessions:** Secure, stateless access to career data.
- **OAuth 2.0 Integration:** Seamless identity synchronization with Google and GitHub using Passport.js.
