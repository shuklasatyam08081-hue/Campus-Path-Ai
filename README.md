**CampusPath AI** is a high-octane career acceleration platform designed to transform developers into industry titans. Powered by **Gemini AI**, it architecturally maps your learning trajectory, verifies your technical DNA via GitHub, and builds elite portfolios that demand attention.

---

## ✨ Elite Features

### 🗺️ Neural Roadmap Architect
- **AI-Driven Trajectory:** Personalized learning paths generated by Gemini AI based on your target role and current expertise.
- **Cyber-Path Visualization:** Interactive neural paths with glowing connections and pulsing milestones.
- **Verified Progress:** Automated repository auditing to verify task completion and unlock achievements.

### 🎨 Identity Studio (Portfolio Builder)
- **Live Preview Engine:** Build your professional legacy in a real-time, studio-grade editor.
- **Elite Templates:** Choose between **Matrix-Cyber** (Neon), **Silicon-Minimal** (Pro), and **Bento-Modern** (Grid) layouts.
- **DNA Integration:** Automatic injection of your GitHub stats, roadmap progress, and verified projects into your public profile.

### 🐙 GitHub DNA Analyzer
- **Codebase Auditing:** Deep-dive analysis of your GitHub repositories to identify technical patterns and skill vectors.
- **Neural Heatmaps:** Visualize your contribution frequency with a custom, high-impact aesthetic.
- **Skill Radar:** Professional-grade visualization of your technical proficiency across languages and frameworks.

## 🛠️ Technical DNA

### Frontend
- **React 19 & Vite:** Lightning-fast, modern reactive UI.
- **Tailwind CSS v4:** Cutting-edge styling with a custom "Cyber-Premium" design system.
- **Framer Motion:** Liquid page transitions and high-performance micro-animations.
- **Lucide React:** Precision-engineered iconography.

### Backend
- **Node.js & Express:** Scalable neural hub for API management.
- **MongoDB & Mongoose:** High-speed vector storage for user identities and roadmaps.
- **Gemini AI SDK:** Large Language Model integration for intelligent roadmap generation.
- **Socket.io:** Real-time event streams for interactive features.

## 🚀 Quick Launch Sequence

### 1. Repository Setup
```bash
git clone https://github.com/Shubham-k-yadav/project.git
cd project
```

### 2. Neural Hub (Backend) Setup
```bash
cd backend
npm install
# Create a .env file with:
# PORT=5000
# MONGO_URI=your_mongodb_uri
# GEMINI_API_KEY=your_key
# JWT_SECRET=your_secret
npm run dev
```

### 3. Interface (Frontend) Setup
```bash
cd frontend
npm install
npm run dev
```
